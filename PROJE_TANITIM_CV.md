

**AutooR - Araç Kiralama Web Platformu** (2026)

Almanya pazarı için geliştirilmiş full-stack araç kiralama platformu. Tarih ve lokasyon bazlı araç arama, rezervasyon yönetimi ve ödeme işlemleri içeren e-ticaret uygulaması.

**Teknolojiler:** Node.js, Express.js, PostgreSQL, JavaScript, Bootstrap 5, Passport.js, Google OAuth, PayPal API, Docker

**Özellikler:**
- RESTful API ile modüler backend mimarisi
- JWT ve Google OAuth ile güvenli kimlik doğrulama
- Admin paneli ile araç, rezervasyon ve kullanıcı yönetimi
- Responsive tasarım ve dinamik fiyatlandırma sistemi

**AutooR - Fahrzeugvermietungs-Webplattform** (2026)

Full-Stack-Fahrzeugvermietungsplattform für den deutschen Markt. E-Commerce-Anwendung mit datums- und standortbasierter Fahrzeugsuche, Reservierungsverwaltung und Zahlungsabwicklung.

**Technologien:** Node.js, Express.js, PostgreSQL, JavaScript, Bootstrap 5, Passport.js, Google OAuth, PayPal API, Docker

**Funktionen:**
- Modulare Backend-Architektur mit RESTful API
- Sichere Authentifizierung mit JWT und Google OAuth
- Admin-Panel für Fahrzeug-, Reservierungs- und Benutzerverwaltung
- Responsives Design und dynamisches Preissystem
